# Earth

Try [live demo](https://disordered-earth.herokuapp.com/) or clone repo.

### How to run:
 - `npm i`
 - `sudo npm run gulp` (without sudo on Windows)
 - If browser didn't open automatically, console will show you address to open
 - Enjoy!
